FancyHolograms

FancyHolograms is a simple and lightweight hologram plugin for minecraft servers using the display entities.
The plugin is blazingly fast because of using packets.
The plugin is only supported for 1.19.4 - latest version
It is highly recommended to use Paper or a fork of it - Folia is also supported.
PlaceholderAPI and MiniPlaceholders is supported.

Features

With FancyHolograms you can create fancy holograms that take advantage of the display entities. You are able to create text, item and block holograms.

Properties you can modify:

  • scale

  • billoard

  • rotation (yaw & pitch)

  • background color

  • text shadow

  • text alignment

  • placeholders in the text

  • and more ...

The holograms are directly sent to the player via packets, which makes FancyHologram very fast and flexible.
